## üåê Platform Support

SSP Server supports multiple platforms with automated deployment:

- **Ubuntu/Debian Linux** - Full systemd integration
- **DigitalOcean Droplets** - Optimized for `ubuntu-s-2vcpu-4gb` configuration
- **macOS/Darwin** - Complete launchd support
- **CentOS/RHEL** - Coming soon

## üåä DigitalOcean Ready

Deploy SSP Server on DigitalOcean with minimal configuration:

```bash
# One-command deployment on your droplet
cd /opt && bash <(curl -sSL https://raw.githubusercontent.com/sspserver/deploy/refs/heads/main/standalone/install.sh)
```

**Recommended**: `ubuntu-s-2vcpu-4gb` droplet (2 vCPUs, 4GB RAM, 80GB SSD)
